Hover over Tooltip

Tooltip, a graphical UI element whereby you can hover over a screen element or component and a textbox pops up containing the infomation on that element. Tooltips following the cursor is my favourite UI element in which adds more interactivity to the User Experience.

You can use different mouse events to position the tooltip relative to the cursor. ClientX and Y is a very useful method only for pages that does not scroll the page down. The name of each element can be accesed using the mouse event, and the target method and the get attribute to obtain the id of the element.

Mousemove works best for mouse tooltips, as it continues the callback as the event is a continuous listener compared to the mouseover, which only listens for the mouse as soon as it enters the element then stops.
